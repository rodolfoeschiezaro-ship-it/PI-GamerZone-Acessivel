📌 Sobre o Projeto
O GamerZone Acessível é um e-commerce desenvolvido com foco em acessibilidade digital para o público gamer com deficiência.
O projeto foi dividido em duas etapas:
1️⃣ Planejamento e Estruturação
2️⃣ Modelagem e Implementação do Sistema
